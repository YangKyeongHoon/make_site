# ✨ 내 로컬 포트폴리오 사이트 (Python) ✨

안녕하세요! 제 포트폴리오를 로컬에서 직접 확인하실 수 있는 웹사이트 프로젝트입니다. 🚀 이 프로젝트는 파이썬을 기반으로 제작되어 여러분의 컴퓨터에서 쉽고 빠르게 실행할 수 있어요! 제 작업물과 기술 스택을 소개하기 위해 만들었어요. 함께 살펴봐요! 💖

---

## 🌟 프로젝트 소개

이 포트폴리오 사이트는 제가 어떤 프로젝트를 했고, 어떤 기술들을 활용했는지 보여주기 위해 만들어졌어요. 개인 프로젝트나 학습 내용을 정리하는 데 아주 유용하답니다! 여러분도 자신만의 포트폴리오를 만들 때 참고하실 수 있을 거예요. 😊

## 💡 주요 기능

*   **다양한 프로젝트 소개**: 제가 진행했던 프로젝트들을 깔끔하게 정리해서 보여줘요.
*   **기술 스택 시각화**: 어떤 기술들을 사용했는지 한눈에 볼 수 있도록 구성될 예정이에요. 📚
*   **간편한 실행**: 파이썬으로 만들었기 때문에 로컬 환경에서 아주 쉽게 실행할 수 있어요!
*   **반응형 디자인 (예정)**: 다양한 기기에서 잘 보이도록 반응형으로 만들 계획이에요. 📱💻

## 🛠️ 기술 스택

이 프로젝트는 다음 기술들을 활용할 예정이에요.

*   **Python**: 백엔드 로직 및 웹 서버 구동
*   **Flask** 또는 **Django**: 웹 프레임워크 (선택 예정)
*   **HTML/CSS/JavaScript**: 프론트엔드 개발
*   **SQLite**: 간단한 데이터 저장 (필요시)

## 🚀 시작하는 방법

이 프로젝트를 여러분의 로컬 환경에서 실행하려면 몇 가지 단계만 거치면 됩니다! 정말 간단해요. 💪

### 1. 레포지토리 클론하기

먼저, 이 프로젝트를 여러분의 컴퓨터로 가져와야 해요. 터미널을 열고 다음 명령어를 입력해주세요.

```bash
git clone [이 프로젝트의 Git 주소]
cd make_site
```

### 2. 가상 환경 설정 (권장)

프로젝트마다 의존성 관리를 위해 **가상 환경**을 사용하는 것이 좋아요. 다음 명령어로 가상 환경을 만들고 활성화해주세요.

```bash
python -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate  # Windows
```

### 3. 필요한 패키지 설치

가상 환경을 활성화했다면, 프로젝트에 필요한 파이썬 패키지들을 설치해야 해요. `requirements.txt` 파일에 목록이 있을 거예요.

```bash
pip install -r requirements.txt
```

### 4. 사이트 실행하기

이제 모든 준비가 끝났어요! 다음 명령어로 포트폴리오 사이트를 실행할 수 있습니다.

```bash
python app.py  # 또는 프로젝트의 메인 실행 파일명
```

성공적으로 실행되면 터미널에 `http://127.0.0.1:5000` (또는 다른 포트)과 같은 주소가 나타날 거예요. 웹 브라우저를 열고 해당 주소로 접속하면 제 포트폴리오 사이트를 볼 수 있습니다! 🎉

## 📝 사용법

사이트가 실행되면 웹 브라우저에서 제 프로젝트 목록을 탐색하고, 각 프로젝트의 상세 설명을 볼 수 있을 거예요. 제 작업물들을 자유롭게 둘러봐 주세요! 🧐
